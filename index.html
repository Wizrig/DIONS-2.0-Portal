Get Started

Build from source.

C++20, cmake, and a handful of dependencies. 13 binaries ready in minutes.

# Clone and build
cd build
cmake --build . -j$(nproc)

# 13 binaries built:
# dions2d (validator), dions2-light (light client), dions-cli (wallet),
# dions-explorer, dions2-migrate, dions2-dns-seeder, dions-relayer,
# + 7 test binaries
