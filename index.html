Architecture

Built different.
Built to last.

22 subsystems working in concert. Every component designed for the next decade of decentralized computing.

Dual Execution Zones

Run EVM and SVM smart contracts side by side. Add new execution zones without a hard fork.

EVM for Solidity | SVM for Rust/BPF programs

Post-Quantum Cryptography

Future-proof signatures with NIST-approved Dilithium, Falcon, and Kyber. Ready before quantum computers arrive.

ECDSA legacy compat + PQC hybrid mode

W3C DID Identity

Native decentralized identifiers following the W3C DID Core 1.0 spec. Aliases, agents, and device delegation built in.

did:dions: namespace | Agent + Device DIDs

Encrypted Messaging

AES-256-CBC + RSA-4096 encrypted messages and private channels. On-chain, censorship-resistant communication.

Direct messages | Group channels | Shadow messages

Ethereum Bridge

Native bridge with relayer consensus and HTLC atomic swaps. Move assets between IOCoin and Ethereum trustlessly.

HTLC atomic swaps | Relayer consensus

Data Availability

Reed-Solomon erasure coding with 64+32 shards. 99.99% data availability sampling guarantee.

64+32 shards | DA sampling built-in

Light Client

Headers-only sync in ~5 minutes. Earn 0.5 IOC per block by running a light node. Under 100 MB storage.

5 min sync | 0.5 IOC block reward

Enhanced PoS Consensus

30-second blocks with checkpoint finality. Validator slashing for double-sign, jailing, and stake delegation.

67% finality | Slashing + jailing

Proof of Participation

Zero-balance nodes earn 0.001 IOC per block by relaying transactions. The first blockchain where anyone can earn from nothing.

Zero balance entry | Relay rewards

Build from source.

C++20, cmake, and a handful of dependencies. 13 binaries ready in minutes.

# Clone and build
cd build
cmake --build . -j$(nproc)

# + 7 test binaries
