Architecture

22 subsystems. One protocol.

Every component designed for the next decade of decentralized computing.

Dual Execution Zones Core

Run EVM and SVM smart contracts side by side. Add new execution zones without a hard fork.

EVM Solidity · SVM Rust/BPF · 11,000+ combined TPS

Post-Quantum Cryptography Security

NIST-approved Dilithium, Falcon, and Kyber. Hybrid mode runs PQC alongside ECDSA.

Dilithium signatures · Falcon compact · Kyber key exchange

W3C DID Identity Identity

Native decentralized identifiers following W3C DID Core 1.0. Aliases, agents, and device delegation.

did:dions: namespace · Agent DIDs · Device delegation

Encrypted Messaging Privacy

AES-256-CBC + RSA-4096 encrypted messages and private channels. On-chain, censorship-resistant.

Direct messages · Group channels · Shadow messages

Ethereum Bridge Interop

Native bridge with relayer consensus and HTLC atomic swaps between IOCoin and Ethereum.

HTLC atomic swaps · Relayer consensus · Multi-sig

Data Availability Storage

Reed-Solomon erasure coding with sharded storage. 99.99% data availability guarantee.

64+32 erasure shards · DA sampling · On-chain file storage

Light Client Access

Headers-only sync in ~5 minutes. Earn 0.5 IOC per block running a light node. Under 100 MB.

5 min sync · 0.5 IOC block reward · 100 MB storage

Enhanced PoS Consensus Consensus

30-second blocks with checkpoint finality. Validator slashing for double-sign, jailing, and delegation.

67% finality · Slashing + jailing · Stake delegation

Proof of Participation Inclusive

Zero-balance nodes earn 0.001 IOC per block by relaying transactions. Earn from nothing.

Zero balance entry · Relay rewards · 0.001 IOC/block
